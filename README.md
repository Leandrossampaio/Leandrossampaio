# Olá, eu sou o Leandro Sampaio 👋

Sou engenheiro de Computação e Elétrica, técnico em Eletrônica e técnico em Informática. Trabalho na interseção entre software, hardware e automação, transformando necessidades práticas em soluções digitais confiáveis — do circuito ao código.

> *"Se eu tivesse seis horas para derrubar uma árvore, passaria as primeiras quatro afiando o machado."* — Abraham Lincoln

## Em que estou trabalhando

- Sistemas web e mobile voltados a operações reais.
- Automação de processos, vendas e comunicação.
- Integração entre aplicações, dispositivos físicos e eletrônica embarcada.

## Projetos em destaque

- **[Monitor Claude](https://github.com/Leandrossampaio/monitor-claude-releases)** — aplicativo Windows para acompanhar o uso do Claude, com alertas, atualizações e instalador público. O código-fonte é privado.
- **PedidoPro** — sistema privado de pedidos para automação de vendas e mensagens, desenvolvido em Python.
- **Sistema de Manutenção** — sistema privado de gestão de manutenção, construído com TypeScript e Supabase.
